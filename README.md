# HR analysis-Dashboard
This project focuses on analyzing employee data to understand attrition trends, workforce distribution, and employee behavior using Power BI. The dashboard provides insights into key HR metrics such as employee count, attrition rate, and job satisfaction to support better decision-making.

## Business Questions (Based on KPIs & Charts)   
1.	What is the total number of employees in the organization?    
2.	How many employees have left the company (attrition)?    
3.	What is the attrition rate (%)?    
4.	How many employees are currently active?    
5.	What is the average age of employees?    
6.	Which department has the highest attrition?    
7.	Which age group has the highest number of employees?    
8.	How does job satisfaction vary across different roles?    
9.	Which education field has the highest attrition?    
10.	Is there any difference in attrition based on gender?    
________________________________________
## 🛠️ Process (Step-by-Step)   
1.	Data Collection     
   o	HR dataset containing employee details (age, department, education, etc.)    
2.	Data Cleaning     
   o	Handled missing values    
   o	Removed duplicates 
   o	Standardized column names 
3.	Data Transformation    
   o	Created calculated columns (Attrition Rate, Active Employees)    
   o	Grouped data by department, age group, and education    
4.	Data Visualization (Power BI)     
   o	Created KPI cards for key metrics    
   o	Used bar charts, pie charts, and line charts    
   o	Applied filters and slicers for better interaction    
5.	Dashboard Design     
   o	Organized layout for clarity 
   o	Used consistent colors and formatting

## 🔍 Project Insights     
•	Sales department has the highest attrition, indicating possible workload or satisfaction issues    
•	Most employees fall in the 25–34 age group, showing a young workforce    
•	Job satisfaction varies across roles, highlighting areas for HR improvement    
•	Employees from Life Sciences background show higher attrition    
•	Gender-based analysis reveals variation in attrition trends    
________________________________________
## 🎯 Final Conclusion    
This dashboard helps in understanding key workforce patterns and highlights areas where the company can improve employee retention. By analyzing attrition, age distribution, and job satisfaction, organizations can take data-driven decisions to improve employee engagement and reduce turnover.
